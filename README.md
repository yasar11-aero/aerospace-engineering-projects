# Aerospace Engineering Projects

This repository contains MATLAB-based numerical methods implementations and aerospace engineering computations developed during my undergraduate studies in Aerospace Engineering.

## Scope

The repository includes:

- Root finding algorithms (Bisection, Newton-Raphson, Fixed Point Iteration)
- Numerical linear algebra methods (Gauss Elimination, LU Decomposition)
- Iterative methods for linear systems (Jacobi, Gauss-Seidel)
- Eigenvalue computation (Power Method)
- Interpolation methods
- Engineering-oriented computational analysis
- Basic aerodynamic calculations (Reynolds number, Lift estimation)
- Future aerodynamic and CFD studies

## Technical Stack

- MATLAB
- Basic Aerodynamics
- Numerical Analysis
- Engineering Mathematics
- Computational Methods

This repository serves as a structured archive of my engineering computations and numerical method implementations, with a focus on aerospace-related applications.
